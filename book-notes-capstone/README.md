# Personal Book Notes Tracker

A full-stack Node.js & Express application built for the Angela Yu Web Development Bootcamp Capstone project. It persists read records, notes, and star ratings using a local PostgreSQL database with a custom glassmorphism midnight aesthetic.

## Installation & Setup

1. Clone this repository to your machine.
2. Initialize database schema tables using the queries logged in `queries.sql`.
3. Install project software dependencies:
   ```bash
   npm install
   ```
4. Start the application backend environment:
   ```bash
   node index.js
   ```
5. View the dashboard at `http://localhost:3000`.
